# DBKK Super App — Prototype

A functional frontend prototype of the Dewan Bandaraya Kota Kinabalu (DBKK) Super App, showcasing the integration of:

- **Sabah Smart Parking** — Pay parking, check compounds, top up credit
- **DBKK iCare** — Submit and track public complaints
- **DBKK e-Services** — Assessment tax, business licence, compound check

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project
3. Import the GitHub repo
4. Deploy (zero config needed — Vercel auto-detects Next.js)

## Built With
